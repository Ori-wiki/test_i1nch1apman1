# INCHAPIN_test

Лендинг жилого комплекса INCHAPIN_test, собранный на Next.js App Router.

## Стек

- Next.js 16
- React 19
- TypeScript
- SCSS Modules
- React Hook Form + Zod
- Framer Motion

## Запуск

Установить зависимости:

```bash
npm install
```

Запустить dev-сервер:

```bash
npm run dev
```

Открыть в браузере:

```text
http://localhost:3000
```

## Команды

```bash
npm run dev
npm run build
npm run start
npm run lint
```

## Перед деплоем

```bash
npm run lint
npm run build
```
