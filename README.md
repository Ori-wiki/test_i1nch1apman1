# INCHAPIN_test

Лендинг жилого комплекса INCHAPIN, собранный на Next.js App Router.

## Демо и деплой

Посмотреть рабочую версию можно по ссылке:

https://test-i1nch1apman1.vercel.app/

## Обзор проекта

Проект показывает современную маркетинговую страницу с:

- Next.js 16 и React 19
- TypeScript для безопасности типов
- SCSS-модулями для компонентных стилей
- Плавным скроллом через `smooth-scrollbar`
- Валидацией формы с помощью `react-hook-form` и `zod`
- Анимациями на `framer-motion`

## Что включено

- Адаптивный сектор с фоном
- Блок проекта с изображением, описанием и видео
- Прилипающая шапка с мобильным меню и селектором квартир
- Контактная форма с проверкой данных и ссылкой на политику конфиденциальности
- Кастомный скролл на рабочем столе и нативный скролл на мобильных


## Быстрый старт

Установите зависимости:

```bash
npm install
```

Запустите dev-сервер:

```bash
npm run dev
```

Откройте приложение в браузере:

```text
http://localhost:3000
```

## Доступные команды

```bash
npm run dev    # запуск dev-сервера
npm run build  # сборка для продакшена
npm run start  # запуск продакшен-сервера
npm run lint   # запуск ESLint
```

## Рекомендуемые проверки перед отправкой

1. Запустите линтер и исправьте ошибки:

```bash
npm run lint
```

2. Проверьте сборку:

```bash
npm run build
```

3. Протестируйте приложение на десктопе и мобильных, особенно поведение скролла.

## Примечания

- Проект использует локальные шрифты из `public/fonts`
- `smooth-scrollbar` отключён на маленьких экранах для сохранения нативного жеста прокрутки
- Контактная форма пока что имитирует отправку на стороне клиента
